Welcome to Crowd-Sourced Accident Reporting Application

A web application in which users can report man-made disasters like road accidents, fire accidents, chemical hazards, etc. The user uploads images of the incident and provides basic details about it, including the location of the incident (tracked using GPS). The images are verified to determine whether they are real or AI-generated. Further, the image is analyzed to identify the type of accident (fire, water, road accidents, etc.). The report is sent to the respective departments with images and location.
